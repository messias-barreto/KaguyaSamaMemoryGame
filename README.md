# KaguyaSamaMemoryGame
Projeto desenvolvido com a finalidade de estudar React-native

## Sobre o Projeto
Jogo da memória inspirado no Anime Kaguya-Sama, com funcionalidades como: 
- reiniciar a partida 
- trocar música selecionada 
- stop música
